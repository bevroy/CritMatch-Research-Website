# CritMatch Branded Website

A GitHub/Codespaces-ready React + Vite landing page for CritMatch, styled to align with the Elionyx Health brand family.

## Brand Direction

- Parent brand: Elionyx Health
- Product: CritMatch
- Visual style: dark navy healthcare-tech, teal/cyan accents, subtle gold accent
- Positioning: EHR-integrated clinical trial cohort matching

## Run in GitHub Codespaces

Open the terminal and run:

```bash
npm install
npm run dev
```

Then open the forwarded port shown by Codespaces.

## Build for Deployment

```bash
npm run build
```

Deploy the generated `/dist` folder to Netlify, Vercel, or GitHub Pages.

## Suggested edits before launch

1. Replace `contact@elionyxhealth.com` with the final public email.
2. Add the final CritMatch logo files to `/public/assets`.
3. Add screenshots or mockups of the actual CritMatch app once available.
