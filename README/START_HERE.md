# CritMatch Website Package

This package contains the marketing content and starter structure for the CritMatch website.

## Recommended GitHub structure
- `/content/` → all website copy
- `/assets/` → logo placeholders + image guidance
- Upload this folder to your GitHub repo
- Deploy via GitHub Pages, Netlify, or Vercel
